# FindAPs

一款用于批量采集接入层交换机下联设备MAC及端口信息的工具系统，同时支持SNMPv2/Telnet/SSH三种采集协议，支持监测接入交换机在线状态，已适配华为(HUAWEI)、新华三(H3C)、锐捷(Ruijie)、寰创(GBCOM)品牌的交换机。

## 功能特性

### 交换机配置
- 支持批量添加交换机设备，分别配置名称、管理IP地址、用户名和密码、Community、协议端口号、品牌等
- 支持WebSSH、Ping监测、批量删除、批量更新、批量采集功能
- 支持列表和卡片两种展示方式并分页

### 采集结果查询
- 支持对全部交换机或对指定交换机查询下联设备MAC地址、端口名称信息
- 支持导出和删除查询结果记录功能
- 支持查询时输入多种MAC格式（如AA:BB:CC:DD:EE:FF / AA-BB-CC-DD-EE-FF / AABB.CCDD.EEFF）

### 批量MAC查询
- 支持同时查询多个下联终端MAC地址并导出结果
- 支持查询时输入多种MAC格式（如AA:BB:CC:DD:EE:FF / AA-BB-CC-DD-EE-FF / AABB.CCDD.EEFF）

### 查看拓扑
- 根据所有交换机已知学习的MAC信息生成链路图谱

### 高级功能
- 支持全局采用跳板机SSH隧道查询交换机状态功能（支持Key登录跳板机）
- 未来支持更多功能（如定时采集、对接企业微信告警等）

### 系统日志
- 支持查看所有采集日志和结果
- 支持采集结果类型分类


## 快速开始

### 环境要求

- Windows10及以上版本的操作系统

### 启动应用

1.直接运行exe程序启动后程序会提示输入端口号（默认5000），程序会自动在浏览器中打开 `http://127.0.0.1:<端口>`。

### 命令行参数

```
用法: FindAPs.exe [-a] [-h] [-p PORT] [-4] [-6]

选项:
  -a, --auto-start  注册开机自启动
  -h, --hide        隐藏黑窗口静默运行
  -p, --port PORT   Web监听端口号
  -4                仅监听IPv4
  -6                仅监听IPv6
  --help            显示帮助信息

注: 若不指定-4和-6，或同时指定-4 -6，则同时监听IPv4和IPv6
```

## 使用流程

1. **添加交换机配置**：在「交换机配置」页面添加交换机设备并设置采集模式然后执行批量查询，若需要跳板机执行则需要在高级设置页面开启
2. **查询采集数据**：在「采集结果查询」页面或「批量MAC查询」页面根据需要查询下联设备MAC地址
